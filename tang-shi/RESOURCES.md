# 唐詩 (Tang Regulated Verse) Resources

## Knowledge

- [Anthology: 300 Tang Poems — University of Virginia Etext](https://cti.lib.virginia.edu/frame.htm)
  The learner's anchor text:《唐詩三百首》with Chinese text and Witter Bynner's English renderings.
  Use for: choosing 範詩 (model poems) for every lesson; the canon we measure against.
- [Tool/Corpus: 搜韻 sou-yun.cn](https://sou-yun.cn/)
  The working poet's toolbench: 平水韻 tables, **律詩校驗** (automated regulated-verse checker),
  對仗詞彙 lookup, allusion search, full poem database. Use for: checking every composition —
  this is the primary automated feedback loop for the whole workspace. *(Verified 2026-06-11.)*
- [Corpus: 全唐詩 — Chinese Text Project](https://ctext.org/quantangshi)
  The full ~49,000-poem Tang corpus in 繁體. Use for: exemplars beyond the 300, checking how Tang
  poets actually used a word or allusion. *(Verified 2026-06-11.)*
- [Dictionary: 漢典 zdic.net](https://www.zdic.net/)
  Per-character phonology including 廣韻 fanqie and 平水韻 category (tone class incl. 入聲).
  Use for: single-character 平仄/韻部 lookups when away from 搜韻. *(Site blocks automated
  fetches; long-established reference — re-verify in browser if a lookup seems off.)*
- Book: 王力《詩詞格律》(中華書局)
  The canonical modern manual of 近體詩 prosody: 平仄, the four line types, 黏對, 孤平, 拗救, 對仗.
  Use for: the authoritative ruling whenever web sources disagree. (Find a print or library copy.)
- Book: 喻守真《唐詩三百首詳析》(中華書局)
  Classic annotated edition of the anthology with per-poem 平仄 marks and compositional analysis.
  Use for: studying how each anthology poem actually executes the rules lesson-by-lesson.

## Wisdom (Communities)

- [詩詞吾愛 www.52shici.com](https://www.52shici.com/)
  Large, active Chinese community of practicing classical poets: posting boards, critique, 擂台
  competitions, and built-in 格律檢測 (now with 拗救 support). Use for: submitting finished 律詩
  for real critique — the mission's audition step. *(Verified 2026-06-11.)*
- [r/ClassicalChinese](https://www.reddit.com/r/ClassicalChinese/)
  English-medium community for 文言 and verse composition questions. Use for: discussing Middle
  Chinese phonology and prosody rules in English when that's easier.
